Моя работа состоит из 5 файлов, из которых 3 основных и 2 второстепенными.
Файл index.php отвечает за отображение странички.
Connect_db.php отвечает за подключение к базе данных.
База данных.sql база данных в которой хранятся комментарии.
Файл stayle.css отвечает за внешний дизайн страницы.
И logo_1.gif это логотип.

Для работы данного файла требуется перенести файлы index.php, stayle.css, logo_1.gif и connect_db.php с небольшими правками логина и пароля, а также импортировать файл database.sql в программу администрирования сайтом.
